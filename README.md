# 🧾 Sistema de Declaração de Imposto de Renda em Excel

## 📌 Sobre o Projeto

Este projeto consiste em uma planilha automatizada em Excel desenvolvida para auxiliar no preenchimento e organização de informações para declaração de imposto de renda.

A solução permite registrar dados financeiros do cliente de forma estruturada, automatizada e intuitiva, utilizando recursos avançados do Excel para melhorar a experiência do usuário e reduzir erros manuais.

---

## 🎯 Objetivo

Desenvolver uma ferramenta prática e automatizada que:

- Organize informações financeiras do cliente
- Permita o preenchimento estruturado de rendimentos
- Automatize cálculos relevantes
- Centralize dados em um menu interativo
- Facilite a visualização das informações declaradas

---

## 🛠️ Tecnologias e Recursos Utilizados

- Microsoft Excel
- Validação de dados
- Formatação condicional
- Menu automatizado com navegação interna
- Estruturação e organização de dados
- Lógica aplicada à análise financeira

---

## 📂 Funcionalidades

✔️ Menu principal automatizado  
✔️ Cadastro de informações pessoais do cliente  
✔️ Preenchimento de rendimentos  
✔️ Registro de outras informações financeiras  
✔️ Automatização de cálculos  
✔️ Organização estruturada dos dados  
✔️ Interface amigável e intuitiva
